project initiator

about

The "project initiator" is a simple automation script to run from your command line. It creates a folder in your destined projects folder, creates a readme and commits it to your github account. The script is available for Windows and Shell terminals.

installation/documentation

win/mac

important note

I wouldn't recommend moving the scripts to another folder. For your convenience just keep it in your root directory. If you are a file organization freak and want to move it anyway follow this tutorial.
